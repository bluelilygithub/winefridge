USING IN THE CURAM THEME
------------------------------------------------------------
• crest-white.png is already wired into the site header and footer
  (white crest sits over the dark hero / ink footer).
• To set the favicon: WordPress admin → Appearance → Editor → (or
  Settings → General "Site Icon" in some setups) → upload favicon-64.png
  or favicon-180.png as the Site Icon.
• On cream sections use crest-ink.png; never place crest-white.png on a
  light background (it will disappear).

------------------------------------------------------------
NOTES & HONEST LIMITATIONS  (please read before wider use)
------------------------------------------------------------
1. RASTER, NOT VECTOR. These are PNGs derived from AI-generated art. They
   scale DOWN well but not UP. For print, signage, embroidery, and etching
   on the cabinets you should commission true VECTOR versions (SVG / EPS /
   PDF) of each variant from a designer. The line-art crest is simple enough
   to redraw cleanly in vector.

2. THE FAVICON AT 32px IS BUSY. The full crest has a crown, shield, bottle
   and four letters — too much detail to read at 16–32px. For a crisp small
   icon, have a SIMPLIFIED monogram favicon made (e.g. just "W" or the
   bottle + WIC), which will read far better in a browser tab.

3. THE MASTER FULL LOCKUP still has a background baked in (it is not
   transparent). The clean, transparent, ready-to-use marks are the three
   crest-*.png files. If you want the FULL lockup (with columns + wordmark)
   on transparency, it needs a clean re-export or a vector rebuild.

4. WORDMARK TEXT. The original artwork reads "EST. AU 2024", which is a
   little awkward. Consider "Est. 2024" or "Australia · Est. 2024" when the
   lockup is next rebuilt.

5. COLOUR ACCURACY. crest-ink / crest-brass are recoloured from the crest
   line-art, so they are perfectly clean and on-brand. The gold in the
   original master board carries a textured metallic sheen that does not
   knock out cleanly — that's why the brass version here is a flat brand
   gold rather than a cropped metallic.
